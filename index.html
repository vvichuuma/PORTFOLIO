PROJECTS

TIMELY

Keep Track of Your Tasks

Timely is the application I created as my end Capstone project while I was studying in the bootcamp. I personally believe it is hard for us human beings to keep everything inside our head and remember things that you wanna do or achieve for that particular day, that is the reason I created this appication. It helps the users to organize their tasks for a particular day. You can even plan yourself ahead for the week with the fiveday feature in the application, which makes it more interesting and useful.

FRONT END : HTML , CSS , JAVASCRIPT , JQUERY , VUE JS
BACK END : RUBY , RUBY ON RAILS.
API's : YAHOO WEATHER API

ATM MACHINE MODEL

Keeping transactions safe

This is an all time favourite app that i always wanted to built. It is an atm machine model where an user can log himself in and see the transactions and the money he/she has in the account. The Admin features makes it more interesting where he can log in and see the amount of money/cash that is in the machine and the Key feature is the notification where the admin will be notified if the machine is running out of cash via through the rails mailer gem.

FRONT END : HTML , CSS , JAVASCRIPT , JQUERY , VUE JS
BACK END : RUBY , RUBY ON RAILS.
